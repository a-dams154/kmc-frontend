# ScoreBoard Frontend

Developed with React + Vite + Recharts.

## Getting Started

1. **Install Dependencies** (if not already done):
   ```bash
   cd frontend
   npm install
   ```

2. **Start the Development Server**:
   ```bash
   npm run dev
   ```

3. **Backend Note**: Ensure your backend is running on `http://localhost:4000`.

## Features

- **Dashboard**: High-performance charts and ranking tables.
- **Admin Panel**: Secure CRUD operations for scores.
- **Glassmorphism UI**: Premium dark mode design system.
- **Auth Flow**: JWT-based registration and login.
